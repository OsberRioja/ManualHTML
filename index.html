IR AL FINAL TEXTO CALIFICAR
Nombre del elemento Descripcion uso
div El elemento HTML "div" es el contenedor genérico para el contenido de flujo. No tiene ningún efecto sobre el contenido o el diseño hasta que se le aplica estilo de alguna manera usando CSS (por ejemplo, se le aplica estilo directamente o se aplica algún tipo de modelo de diseño como Flexbox a su elemento principal). El elemento div debe usarse solo cuando ningún otro elemento semántico (como article o nav) sea apropiado.
h1 Los elementos de encabezado implementan seis niveles de encabezado del documento, 'h1' es el más importante, y 'h6', el menos importante. Un elemento de encabezado describe brevemente el tema de la sección que presenta. La información de encabezado puede ser usada por los agentes usuarios, por ejemplo, para construir una tabla de contenidos para un documento automáticamente.
span Es un contenedor en línea. Sirve para aplicar estilo al texto o agrupar elementos en línea. Debe usarse solo cuando ningún otro elemento semántico sea apropiado. 'span' es muy parecido a un elemento 'div', pero 'div' es un elemento a nivel de bloque mientras que 'span' es un elemento en línea.
br El elemento 'br' produce un salto de linea en un bloque de texto. Es usado para escribir poemas o direcciones, cuando la division de lineas sea significativa
wbr El elemento representa una oportunidad de salto de palabra: una posición dentro del texto donde el navegador puede opcionalmente romper una línea, aunque sus reglas de salto de línea no crearían un salto en esa ubicación. Debe usarse para evitar dejar un signo de puntuación al final de la línea, que el lector podría confundir con el final de una URL.
sup El elemento 'superíndice' especifica el texto en línea que se mostrará como superíndice únicamente por motivos tipográficos. Los superíndices generalmente se representan con una línea de base elevada usando texto más pequeño. El elemento 'sup' solo debe usarse por razones tipográficas, es decir, para cambiar la posición del texto para cumplir con las convenciones o estándares tipográficos, en lugar de únicamente con fines de presentación o apariencia.
sub El elemento 'subíndice' especifica el texto en línea que debe mostrarse como subíndice únicamente por razones tipográficas. Los subíndices generalmente se representan con una línea de base más baja usando texto más pequeño. El elemento 'sub' debe usarse solo por razones tipográficas, es decir, para cambiar la posición del texto para cumplir con las convenciones o estándares tipográficos, en lugar de únicamente con fines de presentación o apariencia.
a crea un enlace a otras páginas de internet, archivos o ubicaciones dentro de la misma página, direcciones de correo, o cualquier otra URL. Se usa para redireccionar a paginas externas o internas por medio de un enlace.
body Representa el contenido de un documento HTML. Solo puede haber unn elemento 'body' en todo el documento Se usa para poner todo el contenido de una pagina o documento html
em es el apropiado para marcar con énfasis las partes importantes de un texto. El elemento puede ser anidado, con cada nivel de anidamiento indicando un mayor grado de énfasis. Se usa cuando se precisa simplemente resaltar un trozo de texto como si estuviese en un modo o voz diferente
img El elemnto imagen HTML representa una imagen en el documento Es el elemento usado para insertar imegenes en la pagina html
mark El elemento mark HTML representa un texto marcado o resaltado. Se usa para marcar un texto como referencia o anotación, debido a su relevancia en un contexto particular.
p El elemento HTML 'p' representa un párrafo, Los párrafos son elementos a nivel de bloque y, en particular, se cerrarán automáticamente si se analiza otro elemento a nivel de bloque antes de la etiqueta de cierre '/p'. Los párrafos generalmente se representan en los medios visuales como bloques de texto separados de los bloques adyacentes por líneas en blanco y/o sangría en la primera línea, pero los párrafos HTML pueden ser cualquier agrupación estructural de contenido relacionado, como imágenes o campos de formulario.
style El elemento HTML 'style' contiene información de estilo para un documento o parte de un documento. Contiene CSS, que se aplica al contenido del documento que contiene el elemento 'style'. El elemento 'style' debe incluirse dentro del 'head' del documento. En general, es mejor colocar sus estilos en hojas de estilo externas y aplicarlas usando elementos 'link'.
table El elemento HTML 'table' representa datos tabulares, es decir, información presentada en una tabla bidimensional compuesta por filas y columnas de celdas que contienen datos. El elemento tabla se usa para representar tablas en una pagina o documento HTML
title El elemento HTML 'título' define el título del documento que se muestra en la barra de título de un navegador o en la pestaña de una página. Solo contiene texto; las etiquetas dentro del elemento se ignoran. El elemento 'título' siempre se usa dentro del bloque 'encabezado' de una página.
